# PrismSelf

聚焦性别理论、心理概念与人际关系的自我探索与科普静态网站。

- 项目类型：静态网站
- 在线访问：<https://prismself.pages.dev/>

## 项目简介

PrismSelf 是一个以网页形式呈现的自我探索与科普项目，聚焦性别理论、心理概念与人际关系。项目包含分析图表、专题指南、共鸣刻度、自评量表、关系对话工具和知识图谱式导航，供学习、整理和个人理解辅助使用。

## 内容与功能

- [index.html](./index.html)：知识图谱入口与总导航。
- [Analyses](./Analyses)：主题分析、概念辨析与对照页面。
- [Guides](./Guides)：全景指南、术语解释与科普页面。
- [Bingos](./Bingos)：共鸣刻度与宾果类页面。
- [Scales](./Scales)：量表与自评页面。
- [Tools](./Tools)：关系沟通与个人梳理工具。

## 使用方式

### 本地使用

直接用浏览器打开 [index.html](./index.html)。

### 线上部署

本项目是静态网页，可部署到 GitHub Pages、Cloudflare Pages、Vercel、Netlify 等静态网站托管平台。

## 技术信息

- 主要技术：HTML、CSS、JavaScript。
- 构建方式：无需构建步骤，静态文件可直接托管。
- 站点入口：[index.html](./index.html)。
- 辅助脚本：[scripts](./scripts) 中包含 SEO、OG 图片和更新记录相关脚本。

## 内容边界

- 本项目内容仅供学习、自我探索和个人理解辅助，不构成医疗、心理、法律或其他专业建议。
- 页面中的自评、量表和共鸣结果不能替代正式评估、诊断或专业支持。
- 访问者在本地填写、记录或导出的个人结果不属于本仓库预设内容，其使用方式由访问者自行决定。
