# PrismSelf

PrismSelf 是一个以网页形式呈现的自我探索与科普项目，包含量表、自评工具、概念全景指南、知识图谱式导航等内容。

- 在线访问：<https://prismself.pages.dev/>

## 目录结构

- [index.html](./index.html)：知识图谱入口与总导航
- [Scales](./Scales)：量表与自评页面
- [Guides](./Guides)：全景指南与术语科普页面
- [Analyses](./Analyses)：主题分析与对照页面
- [Bingos](./Bingos)：共鸣/宾果类页面

## 使用

- 本地使用：直接用浏览器打开 `index.html`
- 线上部署：将仓库内容部署到任意静态网站托管（Pages / Cloudflare Pages / Vercel 等）
